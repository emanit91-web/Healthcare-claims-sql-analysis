Healthcare Claims SQL Analysis

Overview

This project analyzes healthcare claims data using SQL to identify trends in claim activity, payment patterns, and data quality.

Tools Used

- SQL
- DB Browser for SQLite
- Excel (CSV export)

Key Analysis

- Claims by status (Paid vs Denied)
- Total paid by claim type
- Average paid by plan type
- Denied claims analysis
- Diagnosis code cost drivers

Key Insights

- Most claims were paid, with some denials across categories
- Medical claims were the primary cost drivers
- Certain diagnosis codes contributed to higher total spend
- Identified a data quality issue (missing diagnosis code)

Files Included

- claims_data.csv → dataset
- healthcare_claims_sql_project.docx → full analysis
- healthcare_claims_analysis.db → database
- sql_queries.sql → SQL queries used
